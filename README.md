# Mentora AI

The AI programming mentor that gets to know you first.

This is Stage 0 of the build — project scaffold only. See `docs/build-plan.md` for the full roadmap.

## Stack
Next.js (App Router) + TypeScript + Tailwind CSS. Auth, database, and AI integration are added in later stages.

## Local setup
```
npm install
cp .env.example .env.local
npm run dev
```
Visit http://localhost:3000
